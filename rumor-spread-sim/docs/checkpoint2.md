# Checkpoint 2 — Estado Atual do Projeto

## Objetivo
O projeto busca simular a propagação de rumores em uma pequena rede social utilizando agentes generativos inspirados em arquiteturas de memória baseadas em LLMs.

## Funcionalidades implementadas até então
- Estrutura inicial do ambiente social;
- Criação de agentes;
- Sistema básico de crenças;
- Propagação de rumores;
- Geração de grafo social;
- Exportação de GIF animado da simulação.
